# Codex Project Instructions

Read this file first in every new session. This file is the entry point; the detailed project memory lives in `.codex/wiki/`.

## Must Follow

- Work is managed in Linear; read `.codex/wiki/linear.md` before selecting or updating issues.
- New issue branches must start from latest `main`; read `.codex/wiki/branching.md` before branching.
- Scope is limited to the active Linear issue; read `.codex/wiki/scope.md` before editing.
- For product or data-model work, read `.codex/wiki/domain.md`, `.codex/wiki/data-model.md`, and `.codex/wiki/product.md`.
