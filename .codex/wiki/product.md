# Product Direction

Tavolo is a personal cooking archive, not a social network.

MVP focus:

- Local-first app.
- No server, login, public feed, likes, comments, or follows.
- Record meals quickly.
- Browse past cooking records through home, calendar, photos, search, and recap surfaces.

Phase order:

1. Basic recording.
2. Calendar.
3. Detail, edit, delete.
4. Search and tags.
5. Basic recap.
6. Cooking world cup.

Keep implementation aligned with the active Linear milestone and issue.
